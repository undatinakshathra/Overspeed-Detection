# Overspeed-Detection
This project is a simple Java-based Overspeed Detection System that processes vehicle data from a CSV file and determines whether a vehicle is exceeding the speed limit.

📌 Features
Calculates time using frame data
Computes speed (km/h) from distance and time
Detects overspeed violations
Handles invalid and incomplete data
Displays results in a clean tabular format
